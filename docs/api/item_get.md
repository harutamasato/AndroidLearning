====================

概要
----

itemを取得するAPI

リクエストパラメータ
--------------------

| パラメータ名 | 必須 | 内容 | 形式 | 備考 |
| ------------ | ---- | ---- | ---- | ---- |
| `id` | ○ | 整数 | id | |


レスポンス
----------

  * **Content-Type:** `application/json`
  * **Type:** Object

API成功時   

```
{
  "id":"2",
  "body":"haruta",
  "created_at":"2017-01-10 18:45:42",
  "updated_at":"2017-05-02 16:29:39"
}
```   


API失敗時   

```
{"result":false}
```
